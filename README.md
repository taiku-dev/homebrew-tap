# homebrew-tap

Homebrew tap for [taiku](https://taiku.live) — collaborative terminal sharing.

## Install

```bash
brew tap taiku-dev/tap

# Desktop app (includes CLI)
brew install --cask taiku-dev/tap/taiku

# CLI only
brew install taiku-dev/tap/taiku
```

## Upgrade

```bash
brew upgrade taiku-dev/tap/taiku
```
